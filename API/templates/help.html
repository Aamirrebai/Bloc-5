API Documentation

Welcome to the API documentation of our Machine Learning application about the Wine Quality.

This API will help you choose a better quality wine.

This criterion allows you to make a quality wine prediction with the characteristics you enter (e.g. Alcohol, Chlorides, pH)

Requests with tempaltes:

Just click on the "predict" button after entering the data
The predict button will use the POST method to communicate with the API.

Example: Enter the data and click on "Predict".

For example, it returns a message 'This wine is of quality: 6.0 / 10'.

requests with python packge :

import requests
response = requests.post("http://localhost:5000/api",
json={ "input": [[6.6,0.16,0.4,1.5,0.044,48,143,0.9912,3.54,0.52,12.4]]})
print(response.json())

Expected output:
{'predictions': [7.0]}

requests with python packge and multiple inputs :

import requests
response = requests.post("http://localhost:5000/api",
json={ "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8], [6.6,0.16,0.4,1.5,0.044,48,143,0.9912,3.54,0.52,12.4]]})
print(response.json())

Expected output:
{'predictions': [6.0,7.0]}

requests with Bash/Curl :

curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' http://localhost:5000/api

Expected output:
{ "predictions": [ 6.0 ] }
